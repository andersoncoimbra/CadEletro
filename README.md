# CADELETRO
Sistema de cadastro de eletrodomésticos

## Tecnologias
- Laravel 10
- PHP 8
- MySQL 8
- Docker
- Node 18
- Vue 3

## Requisitos

- Docker Compose

## Instalação

1. Clone o repositório:

   ```shell
   git clone https://github.com/andersoncoimbra/CadEletro
2. Entre no diretorio do projeto CadEletro
   ```shell
   cd CadEletro
3. Ative os containers via docker-compose
   ```shell
   docker compose up
